## 体检结论(四问)
1. **完整性**:NaN 已用列中位数填补,填补后 nan_fraction=0。
2. **平衡性**:IR 跨度 1.87→70,满足 must-fix#1(宽 IR),既有近平衡也有极端不平衡。
3. **泄漏风险**:无时间/ID 泄漏字段;重采样将严格限制在训练折内(见实验矩阵防泄漏纪律)。
4. **规模**:adult 4.8万较大,实验时**子采样到 ~8000** 控住 CPU 多种子时间;其余全量。

## 处理记录(可复现)
- 二分类化:少数类标签 = 正类(1)。
- 类别特征:pd.get_dummies(drop_first=True)。
- 缺失:列中位数填补。
- 固定随机种子,分层划分。

## 衔接
清洗后数据交跑实验;派生(采样比变体)在训练折内即时生成,不预先落盘以防泄漏。
