# 行为准则 / Code of Conduct

## 我们的承诺 / Our Pledge

为营造开放、友好的环境,我们承诺让每个人参与 Light 的体验不受骚扰,无论年龄、体型、残障、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向如何。

We pledge to make participation in Light a harassment-free experience for everyone, regardless of background or identity.

## 行为规范 / Standards

鼓励的行为 / Encouraged:
- 友善、尊重不同观点
- 接受建设性批评
- 以社区整体利益为重
- 对其他成员保持同理心

不可接受的行为 / Unacceptable:
- 性化的语言或图像、不受欢迎的关注
- 挑衅、侮辱、人身或政治攻击
- 公开或私下骚扰
- 未经许可公开他人隐私信息
- 其他在专业场合不合理的行为

## 执行 / Enforcement

如遇违规行为,可通过 **1833058953@qq.com** 私下举报。维护者有权移除不符合本准则的评论、提交、代码、issue 等,并对不当行为采取临时或永久封禁。所有投诉都会被审阅并妥善处理,举报人身份予以保密。

Report unacceptable behavior to **1833058953@qq.com**. Maintainers will review and act on all complaints, keeping the reporter confidential.
