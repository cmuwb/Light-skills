# 图表诚实性规范

图表最伤公信力的不是丑，而是**误导**——把没那么大的差异画得很大、把不确定的结果画得很确定。审稿人和读者最容易抓的就是这几类。规范对应 `scripts/figure_integrity_lint.py` 的检查项。

## 硬规范（违反需有明确理由）

### 1. 误差与样本量
- 任何带误差棒/误差带的图，**caption 必须注明误差代表什么**：标准差 SD、标准误 SEM、还是 95% 置信区间 CI。三者宽度差很多，不说等于没说。
- **写明 n=**（每组样本量）。n=3 和 n=300 的同样误差棒含义天差地别。
- 小样本（每组 < ~10）**优先散点叠加 / 箱线 / 小提琴**展示原始分布，不要只画 bar——bar+误差棒会掩盖双峰、离群等真实结构（"bar bar plot" 反模式）。

### 2. 坐标轴
- **y 轴截断必须显式标注断点**（波浪线/断轴符号），不允许偷偷从非零起点开始来放大组间差异。展示比例/计数类数据时，y 轴一般应从 0 起；确有理由不从 0 起，要在 caption 说明。
- 坐标轴范围不为夸大或淡化效果而人为收紧/放大。同一组对比图的同类轴应用一致范围。
- 对数轴要明确标注（log scale），别让读者误读为线性。

### 3. 双 y 轴
- **慎用双 y 轴**：两条不同量纲的曲线叠在一张图上，交点和"看起来相关"往往是坐标轴缩放的人为产物，不是真相关。
- 能拆成两个 panel 就拆分；确需双轴，两轴零点和比例要让对比公平，caption 说明。

### 3b. 三维与装饰
- 不用 3D 饼图/3D 柱状图——透视会扭曲面积和高度的视觉比例。
- 删冗余装饰（chart junk），最大化数据墨水比（Tufte 原则）：每一点墨水都该承载数据。

### 4. 图像完整性（image integrity）
- 显微图/凝胶/影像类：不允许选择性删除、拼接、过度调整对比度/亮度造成误导。任何裁剪/拼接要在 caption 标明，并保留原图。
- 调整亮度对比度必须**全图均匀**，不针对局部。

## 与显著性标注的关系
- 显著性星标（*/**/***）要在 caption 给出对应的 p 阈值和检验方法。
- 不"only show the comparison that worked"——该展示的对比组别要完整，不挑能出显著性的那几对。

## 不是死规矩
这些是默认规范，不是绝对禁令。确有正当理由偏离（如领域惯例、空间限制），在 caption 或正文说明理由即可。lint 脚本只是提示风险点，最终判断仍需作者与审稿标准。
