# 新对话启动提示词（R1-R10 复制即用）

## 使用说明

1. **每轮开一个 Opus 4.8 新对话**（Claude Code，工作目录定位到 Light 仓库根）。
2. 粘贴对应提示词前，确认上一轮已按总纲 §5 收尾（PROGRESS 已更新、CI 绿）。
3. 提示词第一行是**对话命名建议**——客户端不能自动改名，请手动设置，列表即可按序号排列。
4. 轮内上下文吃紧时，用文末"轮内交接通用模板"无缝切新对话。
5. 以下 `<LIGHT_ROOT>` 指仓库根（本机当前 `d:\skill\Light`），路径变了就替换。
