- 前置：R1；db01 的 ai_policy 字段定义在 R4.3（本轮做批量）；db06 上量与 R6 实跑联动；palette.json 与 R6.5 联动。
- 背景：审计四章。九库总判断：**库数 9 个够用，不加新库**；缺的是 db05/06/07 的真实卡规模与 db01/02 的方向纵深。核心原则：**按需生长 + 每用必沉 + 拒绝盲目铺量**；每张卡必须有真实来源链接 + 核验日期。
